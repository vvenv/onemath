# 小学奥数学习体系

**七大模块**：计算、几何、数论、应用题、行程、计数、杂题。

---

## 一、 三年级（激发兴趣，接触典型应用题）

**核心目标**：从简单计算过渡到实际应用，培养逻辑思维。

| 题目类型             | 知识点标签               | 说明                                               |
| :------------------- | :----------------------- | :------------------------------------------------- |
| **速算与巧算**       | 计算、凑整、乘法分配律   | 掌握加减法巧算（如分组、凑整），学习去括号和添括号 |
| **鸡兔同笼**         | 应用题、假设法           | 学习用假设法解决经典的“头脚问题”，培养假设思维     |
| **周期问题**         | 杂题、规律、余数性质     | 根据规律循环找周期，利用余数判断位置或状态         |
| **和差倍问题**       | 应用题、线段图法         | 画线段图分析数量关系，解决和倍、差倍、和差问题     |
| **植树问题**         | 应用题、几何模型         | 区分两端种、一端种、两端不种及封闭图形的情况       |
| **年龄问题**         | 应用题、差倍问题         | 抓住“年龄差不变”作为解题突破口                     |
| **数字谜（算式谜）** | 计算、逻辑推理、数字特征 | 根据运算法则和数字特征，通过推理和试填解决竖式空格 |
| **平均数问题**       | 应用题、统计、移多补少   | 理解“总数÷份数=平均数”，掌握基准数法               |
| **盈亏问题**         | 应用题、比较法、对应思想 | 比较分配方案的差，找出对应关系求份数或总数         |
| **归一问题**         | 应用题、单一量           | 先求出“单一量”（如1份的量），再根据条件计算结果    |

## 二、 四年级（承上启下，拓展思维深度）

**核心目标**：引入几何模型、数论基础，从具象思维向抽象逻辑过渡。

| 题目类型                   | 知识点标签               | 说明                                           |
| :------------------------- | :----------------------- | :--------------------------------------------- |
| **行程问题（相遇追及）**   | 行程、路程=速度×时间     | 学习相遇、追及问题的基本公式，画线段图分析     |
| **定义新运算**             | 计算、符号意识、临场学习 | 题目定义新规则，考察临场理解和运算准确性       |
| **等差数列**               | 计算、数列、通项公式     | 求首项、末项、公差、项数，掌握求和公式         |
| **加乘原理与几何计数**     | 计数、加法原理、乘法原理 | 解决“分类”与“分步”问题，数线段、角、长方形个数 |
| **图形的周长与面积**       | 几何、割补法、平移法     | 利用平移、割补求不规则图形周长和面积           |
| **整除特征**               | 数论、整除、余数         | 掌握被2、3、5、9等常见数整除的数的特征         |
| **质数与合数**             | 数论、质因数分解         | 理解质数概念，掌握分解质因数的方法             |
| **最优化问题（统筹规划）** | 杂题、运筹思想、极端情况 | 如烙饼问题、沏茶问题，通过合理安排时间提高效率 |
| **抽屉原理**               | 杂题、组合、最不利原则   | 也叫鸽巢原理，构造抽屉解决“至少保证”类问题     |
| **逻辑推理**               | 杂题、排除法、真话假话   | 运用列表法、假设法分析复杂的条件关系           |

## 三、 五、六年级（体系完善，冲刺综合应用）

**核心目标**：建立完整的知识体系，强化综合分析和抽象建模能力。

| 题目类型                     | 知识点标签                    | 说明                                                   |
| :--------------------------- | :---------------------------- | :----------------------------------------------------- |
| **分数与小数的混合运算**     | 计算、裂项法、换元法          | 掌握分数裂项、整体约分、字母换元等进阶巧算技巧         |
| **牛吃草问题**               | 应用题、设数法、不变量        | 抓住“原有草量”和“草的生长速度”两个不变量               |
| **多次相遇与变速行程**       | 行程、比例应用、数形结合      | 分析多次折返中的路程关系，利用速度比化连比             |
| **立体几何（表面积与体积）** | 几何、三视图、等积变换        | 涉及正方体、长方体拼接切割，不规则容器中水的体积       |
| **圆与扇形**                 | 几何、割补、容斥原理          | 求阴影部分面积（常用容斥原理、旋转法）                 |
| **约数与倍数（短除模型）**   | 数论、最大公约数、最小公倍数  | 利用分解质因数求因数个数、最大公约数与最小公倍数的关系 |
| **同余问题**                 | 数论、余数性质、中国剩余定理  | 掌握余数的三大性质，解决物不知数问题                   |
| **工程问题**                 | 分数应用题、单位“1”、合作效率 | 设总工作量为单位“1”，利用合作效率解题                  |
| **浓度问题（十字交叉法）**   | 百分数应用题、溶质不变        | 利用溶质守恒或十字交叉法求混合浓度                     |
| **不定方程**                 | 整数解、枚举法、数论          | 方程组中未知数多于方程个数，求整数解                   |
| **概率初步（简单概型）**     | 计数、概率、古典概型          | 掷骰子、摸球、抛硬币等等可能事件，用计数方法计算概率   |
| **概率进阶（复合事件）**     | 计数、概率、独立事件          | 多步事件、独立事件概率、条件概率初步                   |

## 汇总：小学奥数七大模块知识树

| 模块          | 主要包含题型                                                                                 |
| :------------ | :------------------------------------------------------------------------------------------- |
| **1. 计算**   | 速算巧算、等差数列、分数裂项、定义新运算、比较大小                                           |
| **2. 几何**   | 周长与面积、立体几何（三视图）、圆与扇形、勾股定理、五大模型（等高、蝴蝶、鸟头、燕尾、相似） |
| **3. 数论**   | 奇偶性、整除、质合数、约数与倍数、同余问题、位值原理                                         |
| **4. 应用题** | 和差倍、植树、年龄、盈亏、平均数、牛吃草、工程、浓度、经济                                   |
| **5. 行程**   | 相遇追及、火车过桥、流水行船、环形跑道、多人多次相遇、比例行程                               |
| **6. 计数**   | 枚举法、加乘原理、排列组合、容斥原理、抽屉原理、几何计数、概率初步                           |
| **7. 杂题**   | 逻辑推理、最值问题、统筹优化、幻方与数阵图、操作与策略                                       |
