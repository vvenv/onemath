# 小学奥数排列组合

该模块属于奥数七大模块中的**计数模块**，通常从三年级开始接触基础，四五年级逐步深化，六年级进行综合应用与竞赛冲刺 。

## 一、 基础篇：加乘原理与枚举

**适用年级：** 三、四年级 | **核心目标：** 建立分类与分步思想，区分排列与组合。

| 题型分类                    | 典型例题说明                                                                                | 年级   | 难度 | 所属模块  |
| :-------------------------- | :------------------------------------------------------------------------------------------ | :----- | :--- | :-------- |
| **枚举法与树形图**          | 如：用1、2、3能组成多少个不同的两位数？通过树形图或列表不重不漏地列出所有可能 。            | 三年级 | 基础 | 计数      |
| **加法原理（分类）**        | 如：从甲地到乙地，火车有3趟，汽车有2趟，共有多少种走法？（**注意**：分类计数，类类相加） 。 | 三年级 | 基础 | 计数      |
| **乘法原理（分步）**        | 如：搭配问题，上衣2件，裤子3条，共有多少种搭配？或常见的“饮食搭配”问题 。                   | 三年级 | 基础 | 计数      |
| **几何计数（线段/角）**     | 如：一条线段上有5个点，共有多少条线段？常配合“标数法”或组合公式 C(n,2) 求解 。              | 三年级 | 基础 | 计数/几何 |
| **简单的排列（数字/排队）** | 如：4个同学排成一排拍照，有多少种站法？涉及简单的 A(n,m) 全排列思想 。                      | 四年级 | 基础 | 计数      |

## 二、 进阶篇：组合技巧与模型

**适用年级：** 四、五年级 | **核心目标：** 掌握限制条件下的排列组合技巧，引入隔板、捆绑等经典模型。

| 题型分类               | 解题技巧标签   | 典型例题说明                                                                                 | 年级   | 难度 | 所属模块  |
| :--------------------- | :------------- | :------------------------------------------------------------------------------------------- | :----- | :--- | :-------- |
| **组合选法问题**       | 直接法、排除法 | 如：从10人中任选3人参加比赛，有几种选法？涉及 C(n,m) 基本组合运算 。                         | 四年级 | 进阶 | 计数      |
| **排队中的相邻问题**   | **捆绑法**     | 如：3个男生和2个女生排成一排，女生必须相邻，有多少种排法？                                   | 五年级 | 进阶 | 计数      |
| **排队中的不相邻问题** | **插空法**     | 如：节目单中要求小品不能连续出场，有多少种安排方法？先排无限制元素，后插空 。                | 五年级 | 进阶 | 计数      |
| **名额分配问题**       | **隔板法**     | 如：把10个相同的苹果分给3个小朋友，每人至少1个，有几种分法？常转化为正整数解问题 。          | 五年级 | 进阶 | 计数      |
| **数字组数问题**       | 特殊元素优先法 | 如：用0-5这6个数字组成没有重复数字的三位数（或奇数/偶数），通常优先考虑首位或尾位限制 。     | 五年级 | 进阶 | 计数      |
| **几何图形计数**       | 对应法、递推法 | 如：数长方形/三角形个数，或最短路径走法（标数法）。例如常见的网格图中求从A到B的最短路径数 。 | 五年级 | 进阶 | 计数/几何 |

## 三、 挑战篇：复杂限制与综合应用

**适用年级：** 六年级（竞赛冲刺） | **核心目标：** 应对多重限制条件，解决分组分配等复杂综合题。

| 题型分类           | 核心考点     | 典型例题说明                                                                                           | 年级   | 难度 | 所属模块  |
| :----------------- | :----------- | :----------------------------------------------------------------------------------------------------- | :----- | :--- | :-------- |
| **分组与分配问题** | 均匀分组去重 | 如：将6本不同的书平均分给3个学生，或分成3堆。需区分是“分组”还是“分配”，注意平均分组除以 A(n,n) 去重 。 | 六年级 | 挑战 | 计数      |
| **多重限制的排列** | 复合条件     | 如：**甲不站排头，乙不站排尾**，或“甲乙丙顺序固定”的定序问题（消序法） 。                              | 六年级 | 挑战 | 计数      |
| **装箱与错排**     | 容斥原理结合 | 如：四个小球放入四个盒子，恰有一个空盒的放法。或著名的“错位排列”（欧拉信封问题） 。                    | 六年级 | 挑战 | 计数/杂题 |
| **环形排列**       | 固定参照物   | 如：5个小朋友围成一圈做游戏，有多少种坐法？(需注意环形排列无首尾之分，公式为 A(n-1, n-1))。            | 六年级 | 挑战 | 计数      |
| **竞赛选拔逻辑**   | 分类讨论     | 如：含特殊人员的选派（甲不上场、乙丙至少去一人等），需结合分类加法原理精细讨论 。                      | 六年级 | 挑战 | 计数/组合 |

## 💡 附：排列组合核心方法论总结

解决复杂的排列组合问题，关键在于选择合适的**策略与模型**，以下是小学奥数阶段必须掌握的几种核心方法 ：

| 方法名称               | 适用场景                             | 操作要点                                                                   |
| :--------------------- | :----------------------------------- | :------------------------------------------------------------------------- |
| **优先安排法**         | 题目中含有**特殊位置**或**特殊元素** | 先排有特殊要求的元素（如0不能放首位），再排其他无限制元素。                |
| **捆绑法**             | 要求某几个元素**必须相邻**           | 将要求相邻的元素视为一个“大元素”参与排列，再考虑“大元素”内部的排列。       |
| **插空法**             | 要求某几个元素**不能相邻**           | 先排好无限制的元素，形成空位，再将不相邻的元素插入空位中。                 |
| **隔板法**             | 相同元素的分配，**每人至少分一个**   | 将物品排成一排，在物品之间的空隙中插入“隔板”进行分组。                     |
| **排除法（逆向思维）** | 题目限制条件多，正面计算复杂         | **总数 - 不符合条件的总数** = 符合条件的数目。                             |
| **消序法（除法）**     | 元素**定序**或**平均分组**           | 若规定某些元素顺序固定，或出现均匀分组，需除以相同顺序的排列数以避免重复。 |
