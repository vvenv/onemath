# 小学奥数幻方与数阵图

幻方与数阵图属于**杂题模块**，是将数字按特定规律填入图形中的填数游戏。根据题型结构，可分为**幻方、辐射型、封闭型、复合型**四大类 。

## 一、幻方

在正方形方格中填入数字，使每行、每列及对角线上的数字和（幻和）相等 。

| 题型                                             | 年级 | 难度 | 所属模块 |
| :----------------------------------------------- | :--- | :--- | :------- |
| 三阶幻方（九宫格）：将1-9填入3×3方格，使幻和为15 | 三   | 基础 | 杂题     |
| 等差数列幻方：用连续奇数或偶数构制三阶幻方       | 四   | 基础 | 杂题     |
| 补全三阶幻方：已知部分数字，填出剩余空格         | 四   | 基础 | 杂题     |
| 四阶幻方：将1-16填入4×4方格，求幻和并填数        | 四   | 进阶 | 杂题     |
| 五阶幻方：将给定数列填入5×5方格构制幻方          | 五   | 进阶 | 杂题     |
| 六阶幻方：构制6×6阶幻方                          | 五   | 进阶 | 杂题     |
| 积幻方：使每行、每列、对角线上的数的积相等       | 六   | 挑战 | 杂题     |
| 六角幻方：将1-19填入六边形网格，使每条直线和相等 | 六   | 挑战 | 杂题     |
| 拉丁方阵：每行每列各数恰好出现一次（含数字推理） | 四   | 进阶 | 杂题     |

## 二、辐射型数阵图

从中心向外辐射的图形（如十字形、米字形），使每条直线上的数字和相等。

| 题型                                                           | 年级 | 难度 | 所属模块 |
| :------------------------------------------------------------- | :--- | :--- | :------- |
| 基础辐射阵：将5个数填入十字形，使横竖三数和相等                | 三   | 基础 | 杂题     |
| 多线辐射阵：将1-7填入辐射状圆圈，使每条线段三数和相等          | 四   | 基础 | 杂题     |
| 中心重复辐射阵：中心数被多条线共用，先求中心数再填其余         | 四   | 进阶 | 杂题     |
| 多线和值辐射阵：将1-13填入，使每条线段上四个数的和相等         | 五   | 进阶 | 杂题     |
| 双重和值约束：将数字填入"七一"等字形，使横竖行数字和满足特定值 | 五   | 进阶 | 杂题     |
| 分数辐射阵：将分数填入辐射型圆圈，使每条直线上的分数和相等     | 六   | 挑战 | 杂题     |

## 三、封闭型数阵图

图形首尾相连形成闭合回路（如三角形、正方形、圆形），使每条边或每个封闭区域内的数字和相等。

| 题型                                                               | 年级 | 难度 | 所属模块 |
| :----------------------------------------------------------------- | :--- | :--- | :------- |
| 三角形三边和：将1-6填入三角形顶点和边中点，使每边三数和相等        | 三   | 基础 | 杂题     |
| 多边形顶点和：将数字填入正方形/五边形/六边形各边，使每边数字和相等 | 四   | 基础 | 杂题     |
| 双圆重叠型：将数字填入两个相交的圆，使每个圆内的数字和相等         | 四   | 进阶 | 杂题     |
| 多圆重叠型：三个圆两两相交，填入数字使每个圆内数字和相等           | 五   | 进阶 | 杂题     |
| 五圆连环阵：五个圆相互交割，填入1-9使每个圆内数字和相等            | 五   | 进阶 | 杂题     |
| 立方体顶点填数：将数字填入立方体顶点，使每个面四个顶点的和相等     | 六   | 挑战 | 杂题     |
| 正方形套阵：将1-8填入，使大小正方形顶点和相等                      | 五   | 进阶 | 杂题     |

## 四、复合型数阵图

同时包含辐射和封闭两种特征的数阵，需同时满足多条规则。

| 题型                                                               | 年级 | 难度 | 所属模块 |
| :----------------------------------------------------------------- | :--- | :--- | :------- |
| 辐射-封闭结合：三角形三边和+过顶点直线和同时相等                   | 四   | 进阶 | 杂题     |
| 多重图形重叠：将数字填入含直线和大圆周的复合图形，使各线各圆和相等 | 五   | 进阶 | 杂题     |
| 含不等号约束：填入数字使所有不等号方向成立                         | 五   | 进阶 | 杂题     |
| 相邻差约束：填入数字使相邻圆圈之差恰好等于指定差值                 | 五   | 进阶 | 杂题     |
| 数阵最值问题：在满足条件基础上求最大/最小和，并给出填法            | 五   | 挑战 | 杂题     |
| 多组图形和相等：多个2×2正方形中四数和相等，求和的最小可能值        | 六   | 挑战 | 杂题     |

## 解题方法速查

| 方法           | 说明                                               |
| :------------- | :------------------------------------------------- |
| **累加法**     | 将所有"幻和"累加，利用每个位置被计算的次数求关键数 |
| **中心数法**   | 辐射阵中先求被多条线共用的中心数（常用公式推导）   |
| **比较法**     | 对比相关幻和的公共部分，通过差异求特定位置数值     |
| **等差数列法** | 三阶幻方可用罗伯法快速填写等差数列（配对法）       |
| **试填与枚举** | 确定关键位置后，枚举可能组合验证剩余空格           |
