# 小学奥数·弦图模型（经典题型）

| 序号 | 题目                                                                                           |  年级  | 难度 | 题型分类     |
| :--: | :--------------------------------------------------------------------------------------------- | :----: | :--: | :----------- |
|  1   | 四个完全相同的长方形拼成一个大正方形，大正方形周长80，中间小正方形周长16，求长方形面积。       | 三年级 | 基础 | 弦图认知     |
|  2   | 两个数的和为13，积为40，求这两个数。                                                           | 四年级 | 基础 | 和积求数     |
|  3   | 弦图中直角三角形面积为30，大正方形面积为169，求中间小正方形面积。                              | 五年级 | 进阶 | 面积逆推     |
|  4   | 赵爽弦图中，大正方形边长13，小正方形边长7，求直角三角形的周长。                                | 五年级 | 进阶 | 勾股求边     |
|  5   | 正方形ABCD中，E在BC上，F在CD上，∠EAF=45°。将△ABE绕点A旋转90°至△ADG，说明BE、DF、EF的数量关系。 | 六年级 | 挑战 | 旋转构造弦图 |
|  6   | 已知两数的平方和为89，和为13，求两数的积。                                                     | 六年级 | 挑战 | 平方和与弦图 |

---

## 难度说明

| 难度 | 说明                                     |
| :--: | :--------------------------------------- |
| 基础 | 直接识别弦图结构，套用面积公式           |
| 进阶 | 需通过面积等式列方程，逆向求解           |
| 挑战 | 需自主构造弦图，或结合旋转、全等综合推导 |

---

## 核心公式速查

| 场景                     | 公式                                  |
| :----------------------- | :------------------------------------ |
| 大、小正方形与三角形面积 | \( c^2 = 4S\_{\triangle} + (b-a)^2 \) |
| 两数和、积、差关系       | \( (a+b)^2 - 4ab = (a-b)^2 \)         |
| 勾股定理（弦图证明）     | \( c^2 = a^2 + b^2 \)                 |
